LockedIn

A LinkedIn clone using Ruby on Rails and React.js, completed with user authentication, profiles, and full CRUD functionality for posts and comments. Additionally, it utilizes a PostgreSQL as the backend database for efficient data storage and retrieval. I'm currently developing Websocket functionality to allow users to connect in real-time.

Visor

Visor is a vanilla JS application that provides a graphical representation, through a user-friendly interface, of over 8000 stocks' performances over various time frames. Additionally, it leverages the AlphaVantage API and TickerSymbol API to provide real-time and accurate data on various stocks. I'm currently developing a beta coefficient calculator and stock-specific news feed feature for a more data driven user interface.
